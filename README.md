## 使い方

### RequireJSなどのようなdefine関数がある場合,

```javascript
require(['migemoge'], function(migemoge){
    // いけるはず, ※まだ試せていない.
    migemoge.exec("hoge");
});
```

### CommonJS, NodeJSのようにmodule.exportをサポートしている場合.

```javascript
    var migemoge = require("./migemoge");
    migemoge.exec("hoge");
```

### 上記の両方が使えない(Webクライントサイド)

```javascript
// HTMLのscriptタグで読み込み後
// ※この場合window.migemogeのみグローバル汚染します.
migemoge.exec("hoge");
```
